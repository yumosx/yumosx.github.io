字符串格式化

在 Python 中 % 左边是字符串的话，右边可以是一个值，表示对应的字符串格式化

key = 'my_var' value = 1.234
formatted = '%-10s = %.2f' %(key, value) print (formatted)
>>> my_var
my_var = 1.23

在 新的 Python 中，我们可以使用 format 方法来格式化字符串

key = 'my_var' value = 1.234
formatted = '{:<10s} = {:.2f}'.format(key, value) print (formatted)
>>> my_var
my_var = 1.23
